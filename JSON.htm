JSON онлайн редактор, просмотр, редактирование и проверка формата JSON онлайн

С помощью этого онлайн-инструмента вы можете легко создавать, редактировать, просматривать и проверять формат JSON. Вы можете использовать обе панели одновременно, чередуя различные доступные виды и режимы.

Инструмент редактирования JSON совместим с большинством доступных веб-браузеров, включая Google Chrome, Mozilla Firefox, Safari, Opera, Microsoft Edge и Microsoft Internet Explorer 11.

Редактор JSOn имеет различные режимы и представления, в том числе представление в режиме дерева, где вы можете добавлять, изменять, перемещать, дублировать и редактировать элементы и значения, сортировать массивы и объекты, преобразовывать JSON с помощью запросов JMESPath, выбирать цвета с помощью динамического селектора цветов , искать и выделять контент и проверять схемы JSON, среди прочего.

В режиме кода вы можете форматировать и сжимать JSON, восстанавливать JSON и проверять схемы JSON.

В текстовом режиме вы можете форматировать и сжимать JSON, восстанавливать JSON и проверять схемы JSON.

В режиме предварительного просмотра вы можете обрабатывать большие документы типа JSON, до 500 МБ, что означает десятки или сотни тысяч строк и элементов. Вы можете преобразовать JSON с помощью запросов JMESPath, отформатировать и сжать JSON, восстановить JSON и проверить схемы JSON.

Вы можете создать новый JSON, вставить существующий JSON или загрузить файл, содержащий JSON. Вы также можете скачать отредактированный или сгенерированный контент, сохранив его в файле JSON.

Первоначально будут загружены две панели: одна с типом кода, а другая с динамическим типом дерева. Вы можете изменить любую из этих двух панелей на нужный вам формат. Содержимое в обоих представлениях всегда будет синхронизироваться автоматически.
